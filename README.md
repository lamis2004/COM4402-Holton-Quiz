How to run
Ensure Python 3 is installed on your system.
Open a terminal in the project folder.
Run the following command:

python quiz.py

Overview
This project is a console-based multiple-choice quiz developed as a Proof of Concept (PoC) for the Holton College Digital Quiz System.
The application presents a series of questions to the user, accepts input from the keyboard, validates the input, and calculates a final score at the end of the quiz.

Version Control
This project was developed using Git with incremental commits to track progress and improvements throughout development.
